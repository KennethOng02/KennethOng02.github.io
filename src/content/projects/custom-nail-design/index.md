## Background

Group project for a User Interface course, building a web app that lets customers personalize nail sets by selecting shapes, colors, patterns, and decorations.

## Tech stack

Built with React for component-based structure, TypeScript for type safety, and Vite for a fast dev loop.

## Team

Built with three teammates: Chih-Yen Lu, Ngan Ha Pham, and Wen-Hsuan Peng.
