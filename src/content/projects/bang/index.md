## Background

Group project for NTNU CSIE's Computer Programming II course (110-2), implementing the BANG! board game as a terminal application.

## What it does

- Playable BANG! card game rendered entirely in the terminal using the `ncurses` library
- Supports a configurable number of players (defaults to 4)
- Built and run via `make` / `./bin/main -p [player_size]`

## Team

Built with two other teammates (Din, Brian) at National Taiwan Normal University.
